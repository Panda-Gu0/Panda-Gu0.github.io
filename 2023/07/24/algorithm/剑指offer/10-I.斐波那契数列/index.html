斐波那契数列

写一个函数，输入 n ，求斐波那契（Fibonacci）数列的第 n 项（即 F(N)）。斐波那契数列的定义如下：

斐波那契数列由 0 和 1 开始，之后的斐波那契数就是由之前的两数相加而得出。

答案需要取模 1e9+7（1000000007），如计算初始结果为：1000000008，请返回 1。

示例 2：

提示：

  • 0 <= n <= 100

解法一

动态规划，自底向上解法思路，每次遍历保留当前遍历项的前两种，并进行迭代更新，但是对于大数需要进行取模，时间复杂度 O(n)，斐波那契数列的增长是指数级的，对于较大数的运算仍然需要耗费较长的时间

解法二

递归优化，普通的递归会超出运算时间，因为对于一些相同的结果仍然进行递归运算会耗费时间和内存，对一些相同的结果可以用数组进行保存，减少重复运算。
